# Pain Inventory Table

| Pain | 빈도 | 고통 | 현재 우회 방법 | 해결 시 효과 |
|------|------|------|----------------|--------------|
| 예: 매일 이력서 버전 관리 혼선 | 매일 | 중 | 파일명에 날짜 접미사 | 시간 절감 + 실수 감소 |
|  |  |  |  |  |
|  |  |  |  |  |

## 사용 방법
- 빈도: 매일 / 매주 / 가끔
- 고통: 상 / 중 / 하 (사용자가 표현한 표현 그대로 매핑)
- 빈도 × 고통이 큰 항목 상위 1~3개를 R1에서 핵심 pain으로 선정한다.
